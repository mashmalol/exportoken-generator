# راهنمای استفاده از ERC-721 Export Token Metadata Generator

این پروژه یک ابزار ساده برای تولید و درون‌ریزی (Export) اطلاعات متادیتای توکن‌های ERC-721 است. با استفاده از این صفحه می‌توانید اطلاعات مرتبط با کالاهای صادراتی را وارد کرده و متادیتای JSON را دریافت کنید.

## چگونه از این ابزار استفاده کنیم

1. پوشه پروژه را باز کنید.
2. فایل `index.html` را در مرورگر خود باز کنید.
3. فرم را تکمیل کنید:
   - `Token Name`: نام توکن صادراتی را وارد کنید.
   - `Token ID`: شناسه عددی توکن را وارد کنید یا با دکمه `Auto` یک شناسه خودکار بسازید.
   - `Description`: توضیح کوتاهی درباره محتوای توکن بنویسید.
   - `HS Code`: کد HS مربوط به کالا را وارد کنید.
   - `Chapter`, `Heading`, `Duty Rate`, `Commodity Description`: در صورت نیاز جزئیات بیشتری درباره کالا وارد کنید.
   - `Exporter Name`: نام صادرکننده را بنویسید.

## هدف این ابزار

این برنامه به شما کمک می‌کند تا برای فرآیندهای صادراتی یا تجارت بین‌الملل، متادیتا آماده کنید و آن را به شکل توکن ERC-721 استفاده کنید. خروجی می‌تواند برای ثبت اطلاعات کالا یا ساخت توکن‌های دیجیتال مرتبط با صادرات استفاده شود.

## نکات مهم

- این ابزار به صورت محلی در مرورگر اجرا می‌شود و نیازی به نصب ندارد.
- اگر می‌خواهید داده را ذخیره کنید، خروجی JSON را دریافت کرده و در فایل ذخیره کنید.
- با توجه به ساختار ERC-721، هر توکن باید شناسه منحصربه‌فرد داشته باشد.

## ساختار پروژه

- `index.html`: صفحه اصلی و تنها فایل پروژه که رابط کاربری و منطق فرم را در خود دارد.

## راهنمای عمومی برای Export Agentها

اگر منظور از «Export Agents» یعنی سازنده‌های خودکار داده برای صادرات، مراحل کلی به این صورت است:

1. اطلاعات مورد نیاز کالا یا محموله را در فرم وارد کنید.
2. شناسه منحصربه‌فرد برای هر توکن یا کالای صادراتی مشخص کنید.
3. خروجی JSON یا فایل متادیتا را تولید کنید.
4. فایل خروجی را دانلود یا در سیستم‌های دیگر وارد کنید.
